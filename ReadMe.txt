Here is requirements to use an app:

1. You need to have whisper preinstalled to use,
please refer to this link in order to found out what do you need to use whisper:
https://github.com/openai/whisper/tree/main

2. If you have troubles with ffmpeg, please refer to this link:
https://stackoverflow.com/questions/73845566/openai-whisper-filenotfounderror-winerror-2-the-system-cannot-find-the-file

3. In order to make library docx work, please use [pip install python-docx] instead of [pip install docx]
